Overview

This project is a legal case and court opinion dashboard that connects to the CourtListener REST API. It allows users to search for court cases, view judicial opinions, explore court information, analyze opinion clusters, and read citation relationships.

About the API

  • Name: CourtListener
  • Base URL: https://www.courtlistener.com/api/rest/v3/
  • Authentication: Not required for this project
  • Response format: JSON

Fun Fact: Some information that seems like it should be in one endpoint is actually split across multiple endpoints, so you have to combine data manually.

API Endpoints

1. /search

Searches legal cases and opinions using keywords, judges, or court names.

Example Search Request
2. /opinions

Retrieves full judicial opinions including author, date, and case text.

Example Opinions Request
3. /clusters

Groups related opinions under a single case record.

Example Clusters Request
4. /courts

Provides information about federal and state courts.

Example Courts Request
5. /opinion-citations

Shows which opinions cite which cases.

Example Citations Request

How This Project Works

  • On main.html, users can search for legal cases using hoverable images.
  • The search page calls /search and shows results.
  • Opinion pages load detailed text using /opinions.
  • Related cases are grouped using /clusters.
  • Court details come from the /courts endpoint.
  • Citation relationships use /opinion-citations.
